# CareerCraft

CareerCraft is a powerful AI-driven resume builder and ATS checker.

The goal of CareerCraft is to provide everyone with free access to a modern professional resume design and enable anyone to apply for jobs with confidence.

## Features

### 1. Resume Builder
CareerCraft's resume builder allows users to create a modern professional resume easily.

- **AI-Powered Optimization**: Automatically refine your bullet points with strong action verbs and impact-focused language.
- **ATS-Friendly Design**: Resumes are designed to be easily parsed by Applicant Tracking Systems.
- **Privacy First**: Your data stays in your browser. We don't store your personal information on our servers.
- **Instant PDF Export**: Generate perfectly formatted PDF resumes in seconds.

### 2. ATS Checker (Resume Parser)
CareerCraft’s second component is the ATS checker. For those who have an existing resume, the checker can help test and confirm its ATS readability.

## Tech Stack
- **Framework**: Next.js
- **Styling**: Tailwind CSS
- **Icons**: Lucide React
- **State Management**: Redux Toolkit
- **PDF Generation**: @react-pdf/renderer

## Getting Started

1. Clone the repository
2. Install dependencies: `npm install`
3. Run the development server: `npm run dev`
4. Open [http://localhost:3000](http://localhost:3000) to see CareerCraft live.
